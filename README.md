# 🏗️ SafeSpan AI

**Government-Ready Infrastructure Emergency Monitoring & Reporting Platform**

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://jitheshkumar2006-safespan-ai.streamlit.app)

---

## 🚀 Features

| Agent | Description |
|-------|------------|
| 🔍 **Crack Detection** | AI-powered crack detection with heatmap visualization |
| 📊 **Multi-Structure Comparison** | Compare 6+ structures with radar charts |
| 🔔 **Alert Center** | Real-time alerts with timeline visualization |
| 💥 **Failure Impact Simulation** | Collapse impact radius, population & economic loss |
| ⏳ **Structural Aging Clock** | Remaining life estimation & degradation projection |
| 🏢 **Government Complaint** | Auto-generated emergency inspection orders with CRS scoring |

## 🛠️ Tech Stack

- **Frontend**: Streamlit
- **AI/ML**: TensorFlow, OpenCV
- **Charts**: Plotly
- **Data**: NumPy, Pillow

## 📦 Setup

```bash
pip install -r requirements.txt
streamlit run app.py
```
